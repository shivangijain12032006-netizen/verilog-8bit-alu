# Verilog 8-bit Arithmetic Logic Unit (ALU)

This project implements an 8-bit Arithmetic Logic Unit using Verilog.  
The ALU supports multiple arithmetic and logical operations and produces a 16-bit output.

## Features

- 8-bit input operands
- 16-bit output result
- 16 operations using opcode control
- Carry flag detection
- Zero flag detection
- Clock synchronized execution
- Enable control signal

## Operations Supported

- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Shift Left
- Shift Right

## Project Structure

src/ → Verilog source code  
testbench/ → simulation testbench  
docs/ → diagrams and documentation  

## Tools Used

- Verilog HDL
- ModelSim / Vivado simulation
