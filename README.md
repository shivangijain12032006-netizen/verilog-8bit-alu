# Verilog 8-bit Arithmetic Logic Unit (ALU)

This project implements an **8-bit Arithmetic Logic Unit (ALU)** using Verilog HDL.  
The ALU performs **16 different arithmetic and logical operations** based on a 4-bit opcode selector and generates a **16-bit output result** along with status flags.

The design demonstrates core digital design concepts such as combinational logic, sequential logic, clock synchronization, and status flag generation.
